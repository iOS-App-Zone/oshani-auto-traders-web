# Oshani Auto Traders Web Application

A modern web application for Oshani Auto Traders - Direct importers of quality motor vehicles from Japan, Thailand and India.

## Features

- Responsive design
- Pre-order vehicles
- Pre-owned vehicles
- Unregistered vehicles
- Service information
- Contact and team pages

## Deployment

This application can be deployed to:
- GitHub Pages (requires public repository or paid GitHub plan)
- Netlify (free, works with private repositories)
- Vercel (free, works with private repositories)
